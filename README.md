# Cognifyz Level 1 – Restaurant Data Analysis

## Overview

This project was completed as part of the Cognifyz Technologies Data Science Internship Program. The project focuses on data exploration, preprocessing, descriptive analysis, and geospatial analysis using a restaurant dataset.

The analysis was performed using Python in Google Colab.

---

# Objectives

The main objectives of this project are:

* Perform data exploration and preprocessing
* Analyze restaurant data using descriptive statistics
* Visualize restaurant distributions and ratings
* Perform geospatial analysis using latitude and longitude
* Gain insights from restaurant trends and customer ratings

---

# Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Folium
